# La Piada - Сайт ресторана

Современный сайт итальянского ресторана La Piada в Санкт-Петербурге.

## Технологии

- React 19
- Vite
- Framer Motion
- Lucide React

## Разработка

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Деплой на GitHub Pages

1. **Создайте репозиторий на GitHub:**
   - Перейдите на https://github.com/new
   - Название: `green28`
   - Сделайте репозиторий публичным
   - НЕ добавляйте README, .gitignore или лицензию

2. **Подключите удаленный репозиторий:**
   ```bash
   git remote add origin https://github.com/YOUR_USERNAME/green28.git
   git push -u origin main
   ```

3. **Настройте GitHub Pages:**
   - Перейдите в Settings → Pages
   - Source: GitHub Actions
   - После первого пуша workflow автоматически задеплоит сайт

4. **URL сайта:**
   - `https://YOUR_USERNAME.github.io/green28/`

Подробные инструкции в файле [DEPLOY.md](./DEPLOY.md)
