# Видео на главном экране (Hero)

Чтобы фон **проигрывался** во всех браузерах (а не только показывал первый кадр), нужен **MP4 (H.264)** и желательно **небольшой размер** — так браузер быстрее загружает и стабильнее воспроизводит.

## Что сделать

1. **Положи сюда файл** с именем **`hero.mp4`** (в эту же папку `public/video/`).

2. **Рекомендуемая конвертация (меньший размер, стабильное воспроизведение):**

   Установи ffmpeg, если ещё нет: `brew install ffmpeg`

   Затем из корня проекта выполни **одну** из команд.

   - **Вариант А — компактный (720p, ~1–2 Мбит/с, быстрая загрузка и надёжный автозапуск):**
     ```bash
     cd /Users/Voronin/Desktop/work/websites/v-pitere-pit
     ffmpeg -i /Users/Voronin/Desktop/work/1.mov -an -c:v libx264 -vf "scale=-2:720" -b:v 1.5M -movflags +faststart -y public/video/hero.mp4
     ```

   - **Вариант Б — только конвертация без сжатия (если исходник уже небольшой):**
     ```bash
     ffmpeg -i /Users/Voronin/Desktop/work/1.mov -an -c:v libx264 -movflags +faststart -y public/video/hero.mp4
     ```

   Параметры в варианте А: без звука (`-an`), высота 720px (`scale=-2:720`), битрейт 1.5 Мбит/с (`-b:v 1.5M`), быстрый старт в браузере (`-movflags +faststart`).

3. **Без ffmpeg:** конвертируй .mov → .mp4 в онлайн-сервисе, по возможности выбери разрешение 720p и «для веб» — затем сохрани как `hero.mp4` в папку `public/video/`.

После этого перезапусти dev-сервер или пересобери проект — видео должно автоматически проигрываться и зацикливаться.
